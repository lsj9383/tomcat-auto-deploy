#Tomcat Auto Deploy
本项目用于tomcat服务器远程自动部署网站应用。
##运行环境
* jdk1.5及其以上版本
* tomcat7及其以下版本，需要配置CATALINA_HOME到环境变量。

##1.*快速开始*
这里用尽量简单的语言描述如何通过命令行自动将本地开发的网站部署到远程主机上。本项目在远程主机和本地都需要进行配置。
###1.*客户端*
在远程主机配置完成的情况下，将webapp部署到远程主机下是非常简单的，只需通过以下指令:
```
java -jar tad-client.jar -uroot -proot -rhttp://192.168.1.2:8080 -atrans
```
如此便可以将本地名为trans的webapp，部署到192.168.1.2:8080的远程主机上.使用的用户名和密码分布是root，这个用户名和密码在远程端配置，与远程主机匹配才可以webapp部署。
